### 6. Screenshots (Required for App Store Connect)

You need to create screenshots showing your app in action:

#### Required Sizes:

**iPhone 6.7" (iPhone 15 Pro Max, etc.)**
- [ ] 1290 x 2796 pixels - 3 to 10 screenshots
- Devices: iPhone 15 Pro Max, 14 Pro Max, 13 Pro Max, 12 Pro Max

**iPhone 6.5" (iPhone 11 Pro Max, XS Max)**
- [ ] 1242 x 2688 pixels - 3 to 10 screenshots
- Can auto-scale from 6.7" screenshots

**iPhone 5.5" (iPhone 8 Plus, 7 Plus, 6s Plus)**
- [ ] 1242 x 2208 pixels - 3 to 10 screenshots

**iPad Pro 12.9" (if supporting iPad)**
- [ ] 2048 x 2732 pixels - 3 to 10 screenshots

#### Suggested Screenshot Content:
1. **Main screen** - Puzzle of the Day with word length selection
2. **Active puzzle** - Showing letter tiles and keyboard
3. **Gameplay** - Mid-puzzle with some letters changed
4. **Completion** - Celebration screen when puzzle is solved
5. **Free Roam** - Free Roam mode interface
6. **Tutorial** - Onboarding overlay explaining features

#### Screenshot Tips:
- Use simulator to capture at exact resolution
- Add text overlays highlighting features (optional but recommended)
- Show the app in use, not just empty screens
- Make first screenshot most compelling (appears in search)
- Keep consistent style across all screenshots

## BUILD & UPLOAD

### 13. Build Preparation

#### Pre-Build Checks
- [ ] All compiler warnings resolved
- [ ] No crashes on launch
- [ ] All features tested on physical device
- [ ] Tested on multiple device sizes (iPhone, iPad if supported)
- [ ] Tested in portrait and landscape orientations
- [ ] Memory usage reasonable
- [ ] Battery usage acceptable

#### Build Configuration
- [ ] Select **Generic iOS Device** (not Simulator)
- [ ] Select **Release** build configuration
- [ ] Product > Archive

#### Archive Validation
- [ ] Archive created successfully
- [ ] Validate Archive (in Organizer)
  - Checks for common issues
  - Verifies signing
  - Validates privacy manifest
- [ ] Fix any validation errors

### 14. Upload to App Store Connect

#### Via Xcode
- [ ] Click "Distribute App" in Organizer
- [ ] Select "App Store Connect"
- [ ] Upload
- [ ] Wait for processing (10-60 minutes)

#### Verify Upload
- [ ] Log into App Store Connect
- [ ] Navigate to your app
- [ ] Check "TestFlight" tab for build
- [ ] Build shows "Processing" then "Ready to Submit"
